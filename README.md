个人开发导航面板
 
项目简介
 
一款纯前端静态开发导航页，无需后端、开箱即用，专为开发者打造。
整合各类浏览器、Linux发行版、编程IDE下载地址，内置Linux常用命令速查（一键复制），支持浅色/深色双主题，自动保存用户偏好，移动端完美适配。
 
功能特性
 
1. 双主题切换
浅色 / 深色模式，配置存入浏览器本地存储，刷新页面自动记忆上次选择。
​
2. 多分类导航聚合
​
- GitHub 快捷入口（主页 + 本项目仓库）
​
- 各类浏览器安装包下载（Chromium、Firefox、Librewolf、Brave等）
​
- 主流Linux发行版镜像下载（Ubuntu、Arch、Kali、Fedora等）
​
- 全平台开发工具（VSCode、Android Studio、Python、Node.js、Lua等）
​
3. Linux命令速查工具
​
- 内置常用运维、Git、Docker、磁盘、权限、系统服务命令
​
- 一键复制功能，复制后按钮临时提示「已复制」
​
- 支持关键词实时过滤搜索命令
​
4. 交互优化
​
- 卡片悬浮上浮+渐变阴影动效
​
- 页面滚动「回到顶部」按钮
​
- 顶部浏览器下载提示栏可关闭，永久记忆关闭状态
​
- 平滑滚动、响应式布局，手机/电脑均可正常浏览
​
5. 轻量化无依赖
仅原生HTML+CSS+JS，无Vue/React等框架，打开秒加载，支持GitHub Pages、静态服务器部署。
